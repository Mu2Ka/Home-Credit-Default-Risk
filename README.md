# Home Credit Default Risk

Пет-проект по кредитному скорингу на данных соревнования Home Credit Default Risk.

Проект находится в разработке и будет постепенно обновляться. Сейчас в репозитории уже есть базовый анализ данных, объединение нескольких таблиц, feature engineering и первые baseline-модели.

## Что внутри

- `home-credit-default-risk/main.ipynb` - основной ноутбук проекта
- `requirements.txt` - зависимости проекта

## Что уже сделано

- загружены и связаны основные таблицы соревнования
- выполнена базовая агрегация внешних источников данных
- добавлены первые инженерные признаки
- построены baseline-модели для предсказания дефолта
- подготовлена база для дальнейших улучшений

## Статус проекта

Проект в активной разработке. В следующих обновлениях можно ожидать:

- более аккуратный feature engineering
- расширенный EDA
- улучшение качества модели
- сравнение нескольких алгоритмов
- более чистую структуру ноутбука и результатов

## Датасет

Данные не загружаются в репозиторий.

Скачать датасет можно здесь:
- [Home Credit Default Risk](https://www.kaggle.com/competitions/home-credit-default-risk)

После скачивания помести файлы `.csv` в папку `home-credit-default-risk/`.

## Технологии

- Python
- pandas
- numpy
- matplotlib
- scikit-learn
- lightgbm
- jupyter

## Как запустить

1. Создать и активировать виртуальное окружение.
2. Установить зависимости:

```bash
pip install -r requirements.txt
```

3. Скачать датасет с Kaggle.
4. Положить `.csv` файлы в папку `home-credit-default-risk/`.
5. Запустить Jupyter Notebook и открыть `home-credit-default-risk/main.ipynb`.
